---
title: "Measures"
description: "Manage security and compliance measures — the controls your organization implements to address risks."
---

Measures are the concrete actions your organization takes to manage risk and maintain compliance. Each measure tracks its implementation state and can be linked to risks, framework controls, tasks, and evidence.

## The Measure object

<ResponseField name="id" type="string" required>
  Global ID (GID) of the measure.
</ResponseField>

<ResponseField name="category" type="string" required>
  Category of the measure (free-form string, e.g. `"Access Control"`, `"Encryption"`).
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the measure.
</ResponseField>

<ResponseField name="description" type="string | null">
  Optional description of the measure.
</ResponseField>

<ResponseField name="state" type="MeasureState" required>
  Current implementation state. See [MeasureState enum](#measurestate-enum).
</ResponseField>

<ResponseField name="created_at" type="string (date-time)" required>
  ISO 8601 timestamp of when the measure was created.
</ResponseField>

<ResponseField name="updated_at" type="string (date-time)" required>
  ISO 8601 timestamp of the last update.
</ResponseField>

## MeasureState enum

| Value | Description |
|-------|-------------|
| `NOT_STARTED` | The measure has not yet been started |
| `IN_PROGRESS` | The measure is currently being implemented |
| `NOT_APPLICABLE` | The measure does not apply to this organization |
| `IMPLEMENTED` | The measure is fully implemented |

## Operations

### listMeasures

Returns all measures for the organization.

**MCP tool:** `listMeasures`

#### Parameters

<ParamField body="organization_id" type="string" required>
  GID of the organization.
</ParamField>

<ParamField body="size" type="integer">
  Number of results per page.
</ParamField>

<ParamField body="cursor" type="string">
  Pagination cursor from a previous response's `next_cursor`.
</ParamField>

<ParamField body="order_by" type="object">
  <Expandable title="order_by fields">
    <ParamField body="field" type="string">
      Sort field. One of: `CREATED_AT`, `NAME`.
    </ParamField>
    <ParamField body="direction" type="string">
      Sort direction. One of: `ASC`, `DESC`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="filter" type="object">
  <Expandable title="filter fields">
    <ParamField body="query" type="string">
      Full-text search query.
    </ParamField>
    <ParamField body="state" type="MeasureState">
      Filter by implementation state.
    </ParamField>
    <ParamField body="category" type="string">
      Filter by category.
    </ParamField>
  </Expandable>
</ParamField>

#### Response

<ResponseField name="measures" type="Measure[]" required>
  Array of measure objects.
</ResponseField>

  Cursor for the next page.
</ResponseField>
