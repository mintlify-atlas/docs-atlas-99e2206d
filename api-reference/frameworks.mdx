---
title: "Frameworks"
description: "Work with compliance frameworks and their associated controls."
---

Frameworks represent the compliance standards your organization is working toward — such as SOC 2, ISO 27001, GDPR, or HIPAA. Each framework contains controls that you can link to your measures and evidence.

## The Framework object

<ResponseField name="id" type="string" required>
  Global ID (GID) of the framework.
</ResponseField>

<ResponseField name="organization_id" type="string" required>
  GID of the organization this framework belongs to.
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the framework (e.g. `"SOC 2 Type II"`, `"ISO 27001"`).
</ResponseField>

<ResponseField name="description" type="string | null">
  Optional description of the framework.
</ResponseField>

<ResponseField name="created_at" type="string (date-time)" required>
  ISO 8601 timestamp of when the framework was created.
</ResponseField>

<ResponseField name="updated_at" type="string (date-time)" required>
  ISO 8601 timestamp of the last update.
</ResponseField>

## Operations

### listFrameworks

Returns all frameworks for the organization.

**MCP tool:** `listFrameworks`

#### Parameters

<ParamField body="organization_id" type="string" required>
  GID of the organization.
</ParamField>

<ParamField body="size" type="integer">
  Number of results per page.
</ParamField>

<ParamField body="cursor" type="string">
  Pagination cursor from a previous response's `next_cursor`.
</ParamField>

<ParamField body="order_by" type="object">
  <Expandable title="order_by fields">
    <ParamField body="field" type="string">
      Sort field. Currently only `CREATED_AT` is supported.
    </ParamField>
    <ParamField body="direction" type="string">
      Sort direction. One of: `ASC`, `DESC`.
    </ParamField>
  </Expandable>
</ParamField>

#### Response

<ResponseField name="frameworks" type="Framework[]" required>
  Array of framework objects.
</ResponseField>

<ResponseField name="next_cursor" type="string">
  Cursor for the next page.
</ResponseField>

<Note>
